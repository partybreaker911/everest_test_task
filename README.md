### HOW TO RUN IT

-   Easy as it can be:

    `$ docker-compose -f local.yml build --no-cache`

    `$ docker-compose -f local.yml up`

-   If you need something from `flask cli` in docker just run something like this:

    `$ docker-compose -f local.yml run web flask db init`
